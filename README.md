# Jayanth Srinivasan - Portfolio

A modern, interactive portfolio website built with React, Vite, and Framer Motion. This site features a glassmorphism design and highlights major professional achievements and technical skills.

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Build Tool**: Vite
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Styling**: Vanilla CSS (Custom Variable System)

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd my_portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create an optimized production build:
```bash
npm run build
```
The output will be in the `dist/` directory.
